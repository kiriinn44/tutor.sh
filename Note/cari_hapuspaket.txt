# hapus paket

Hapus paket dnsrecon:

sudo apt-get remove dnsrecon
- Hapus paket dan file konfigurasi (opsional, untuk pembersihan menyeluruh):

sudo apt-get purge dnsrecon

-Hapus dependensi yang tidak diperlukan lagi:

sudo apt-get autoremove


Metode 2: Jika diinstal melalui pip (Manajer Paket Python)
dnsrecon adalah skrip Python dan sering diinstal menggunakan pip. Coba jalankan perintah ini di Terminal: 

pip uninstall dnsrecon
atau, jika Anda menggunakan pip3:

pip3 uninstall dnsrecon


Metode 3: Jika diinstal secara manual dari repositori Git
Jika Anda mengkloning repositori Git dan menginstalnya secara manual (misalnya menggunakan setup.py atau hanya menjalankan skripnya langsung dari direktori), Anda biasanya hanya perlu menghapus direktori tempat Anda mengkloningnya.
Navigasikan ke direktori induk (misalnya, jika Anda mengkloningnya di ~/Downloads/dnsrecon):

cd ~/Downloads/
Hapus direktori dnsrecon:

rm -rf dnsrecon

#mencaripaket


1. Mencari paket berdasarkan kata kunci (nama dan deskripsi)

apt search dnsutils

apt search dnsutils | less

2. Mencari hanya berdasarkan nama paket:

apt search --names-only dnsutils

3. Mencari paket yang sudah terinstal:

apt list --installed | grep bind9-dnsutils

4. Melihat informasi detail paket:

apt show bind9-dnsutils
