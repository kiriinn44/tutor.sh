aktifasi virtul machine di bios gigabyt b450m

Untuk mengaktifkan virtualisasi (VT) di BIOS Gigabyte B450M, restart PC, tekan tombol Del atau F2 untuk masuk BIOS, lalu ke tab M.I.T. > Advanced Frequency Settings > Advanced CPU Core Settings, temukan SVM Mode (AMD) atau Intel Virtualization Technology (Intel) dan ubah menjadi Enabled, lalu simpan dengan F10 dan keluar. 
Langkah-langkah Mengaktifkan Virtualisasi di BIOS:
Masuk ke BIOS/UEFI:
Restart komputer Anda.
Saat logo Gigabyte muncul, tekan tombol Del atau F2 berulang kali untuk masuk ke BIOS/UEFI.
Navigasi ke Pengaturan CPU:
Gunakan tombol panah (atau mouse jika didukung) untuk pindah ke tab M.I.T. (Motherboard Intelligent Tweaker).
Pilih Advanced Frequency Settings.
Pilih Advanced CPU Core Settings.

Aktifkan Fitur Virtualisasi:
Cari opsi SVM Mode (jika menggunakan prosesor AMD) atau Intel(R) Virtualization Technology (jika menggunakan Intel).
Ubah pengaturannya dari Disabled menjadi Enabled.
Simpan dan Keluar:
Tekan tombol F10 untuk menyimpan perubahan.
Pilih Yes atau OK untuk mengonfirmasi.
Komputer akan restart dengan virtualisasi aktif.
