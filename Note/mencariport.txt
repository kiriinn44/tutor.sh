#Berikut adalah beberapa cara untuk mencari port yang dibuka pada suatu IP:
1. Menggunakan perintah nmap:
- Buka terminal atau command prompt
- Ketik perintah: nmap -sT [IP address]
- Contoh: nmap -sT 216.58.194.174
atau
- nmap -p- [IP address]
- Contoh: nmap -p- 216.58.194.174
atau
- nmap -p 1-65535 [IP address]
- Contoh: nmap -p 1-65535 216.58.194.174
1. Menggunakan perintah netstat (tidak direkomendasikan karena tidak akurat):
- Buka terminal atau command prompt
- Ketik perintah: netstat -an | findstr [IP address]
- Contoh: netstat -an | findstr 216.58.194.174
1. Menggunakan perintah telnet:
- Buka terminal atau command prompt
- Ketik perintah: telnet [IP address] [port]
- Contoh: telnet 216.58.194.174 80
1. Menggunakan perintah nc (Netcat):
- Buka terminal atau command prompt
- Ketik perintah: nc -v -z -w 1 [IP address] [port]
- Contoh: nc -v -z -w 1 216.58.194.174 80
Perlu diingat bahwa beberapa perintah di atas mungkin tidak dapat digunakan pada semua sistem operasi atau perangkat. Selain itu, beberapa perintah mungkin memerlukan hak akses administrator untuk dijalankan.

#Berikut adalah beberapa cara untuk membuka IP dan port yang open:

#Menggunakan Web Browser

1. Buka web browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge.
2. Ketik alamat IP dan port yang open di address bar, contoh: http://216.58.194.174:80
3. Tekan Enter untuk membuka halaman web yang terkait dengan IP dan port tersebut.

#Menggunakan Telnet

1. Buka terminal atau command prompt.
2. Ketik perintah: telnet [IP address] [port]
Contoh: telnet 216.58.194.174 80
3. Tekan Enter untuk membuka koneksi telnet ke IP dan port tersebut.

#Menggunakan Netcat (nc)

1. Buka terminal atau command prompt.
2. Ketik perintah: nc -v -z -w 1 [IP address] [port]
nc -v -z -w 1 216.58.194.174 80
3. Tekan Enter untuk membuka koneksi netcat ke IP dan port tersebut.

#Menggunakan Putty (untuk SSH)

1. Buka Putty.
2. Masukkan alamat IP dan port yang open di kolom Host Name (or IP address).
3. Pilih protokol SSH di kolom Connection type.
4. Klik Open untuk membuka koneksi SSH ke IP dan port tersebut.

#Menggunakan FTP Client (untuk FTP)

1. Buka FTP client seperti FileZilla.
2. Masukkan alamat IP dan port yang open di kolom Host.
3. Pilih protokol FTP di kolom Protocol.
4. Klik Connect untuk membuka koneksi FTP ke IP dan port tersebut.
Perlu diingat bahwa beberapa metode di atas mungkin tidak dapat digunakan pada semua sistem operasi atau perangkat. Selain itu, beberapa metode mungkin memerlukan hak akses administrator untuk dijalankan.
